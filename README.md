# Essência de Mulher PDV — V1

Projeto inicial publicável e funcional em HTML, CSS e JavaScript.

## Como abrir
Abra `index.html` no navegador.

## Login inicial
- Usuário: `admin`
- Senha: `1234`

## Recursos incluídos
- Dashboard no estilo visual aprovado
- Nova venda e nota rápida
- Baixa automática no estoque
- Produtos com código automático por categoria e cor
- Clientes e histórico básico
- Caixa diário
- Financeiro com faturamento, custo, reposição e lucro real
- Contas a receber
- Brindes
- Catálogo responsivo com pesquisa, categorias, carrinho e WhatsApp
- Origem da venda
- Usuários e permissões demonstrativos
- Auditoria
- Backup e restauração local

## Importante
Esta V1 usa `localStorage`, portanto os dados ficam salvos no navegador do aparelho.
Para uso em vários aparelhos e vários funcionários ao mesmo tempo, a próxima etapa deve incluir backend e banco de dados.
